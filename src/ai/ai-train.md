# 星斗智能体 - AI 教育推广比赛展示方案

## 一、产品背景

**场景定位**：学生学习完一个知识点后，老师需要给学生出该知识点的练习题和编程题，以及试卷，使用智能体一键出题解决出题痛点，走通学生练习、考试交卷、自动评分、教师后台查看整个闭环。

**核心价值**：真实场景还原，IDE 环境与AI智能体工具深度融合，助力学生无缝衔接职场。

**技术架构**：三个智能体（星斗插件、星斗智聊、CC-GUI）集 IDE 于一身，根据知识点一键生成练习内容。

---

## 二、特色展示

### 2.1 多种题型支持

| 题型   | 支持情况 |
| ------ | -------- |
| 单选题 | ✅ 支持   |
| 多选题 | ✅ 支持   |
| 填空题 | ✅ 支持   |
| 简答题 | ✅ 支持   |
| 编程题 | ✅ 支持   |

### 2.2 五维度练习模式

1. **强基练习** - 夯实基础知识点
2. **巩固练习** - 强化已学内容
3. **课后作业** - 延伸学习深度
4. **拓展练习** - 挑战高阶能力
5. **随堂测试** - 即时检验效果

### 2.3 一键生成功能矩阵

| 功能                 | 说明                             | 输出形式               |
| -------------------- | -------------------------------- | ---------------------- |
| 一键生成题目和知识点 | 输入知识点名称，自动生成配套题目 | 结构化数据             |
| 一键生成 MD 文档     | 题目 MD 文档放在知识点上展示     | 直接上传项目，直接链接 |
| 一键生成答案解析     | 详细解析可集成到 AI 智能体       | 智能体内部调用         |
| 一键生成 JAVA 代码   | 完整可运行代码                   | 直接上传项目，直接链接 |
| 一键生成测试案例     | 编程题测试用例，支持本地 check   | 本地验证               |
| 一键生成试卷         | 自动组卷、自动判分、支持重答     | 全流程闭环             |
